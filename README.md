# GLUT RUMBLE

Small 3D pokemon-like battle game made using C++, OpenGL(GLUT) and Blender.

Academic project from the 2025 course TCG Computer Graphics by team "Affectionate".

# Gameplay:
Player vs Player ().
Player vs Computer.

# Game controls:
N(p1), M(p2) - choose characters.
Space - start game / next turn / proceed.
1, 2, 3 number keys and left, down, right arrow keys - select character skill/move.
I - toggle computer opponent (let computer take over p2).
O - toggle music.
P - pause / unpause game, reset game speed.
[, ] - decrease / increase game speed.

# Developer controls:
R - toggle axis.
F - toggle freeview / noclip.
WASD - freeview horizontal movement.
Q, E - freeview vertical movement.
Z, X - freeview zoom in and out.
Mouse - freeview rotate camera.
G - toggle godmode (lock health and mana for both player).
F1 - toggle wireframe mode.

# Other information:
Skill selection is discrete, so that other real players can't see what you have selected,
instead, casted skills from each turn will be logged in console.
Changing game speed will be logged in console.
